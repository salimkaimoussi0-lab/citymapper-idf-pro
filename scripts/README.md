# Scripts

Ce dossier contient les scripts d'exécution du projet d'analyse.

## run_colab_style_analysis.sh

Commande principale :

bash scripts/run_colab_style_analysis.sh

Ce script lance toutes les analyses dans le bon ordre.

Il génère les cartes, les tableaux, les fichiers Excel et le rapport global.

## build_report_index.py

Ce script construit le rapport HTML final à partir des cartes et tableaux générés.

Résultat :

reports/rapport_colab_style.html
